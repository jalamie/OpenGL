MAZE GAME

Jamie Goh

Project description

The objective of this game is to destroy all objects found in the scene. The santa is moving along a catmull-Rom Spline and the plane circulating amove the maze. Players can move the shark around with the keyboard and look around with the mouse. The game will only start when the user starts moving and it will trigger a countdown of 100 seconds. The game will get harder as the time passes as the fog will progressively get denser. If all the items are destroyed it will display a banner stating that the player have won. However if the player has not done so then the player would lose and be unable to move. Players are able to customise the viewpoint with the left and right arrows to increase or decrease their field of vision. Players are also able to customise the light, animation and fog with the menu. Players can also customise the size of shark, santa and plane as well of the speed of shark and plane with the config file.
